# 🔥 1. En kritik kontrol: Hangi ürünleri çekiyorsun?

Kodunda büyük ihtimal şuna benzer bir şey var:

### ❌ HATALI (en yaygın hata)

```js
const offerings = await Purchases.getOfferings();
const products = offerings.current.availablePackages;
```

👉 Eğer `current offering` yanlışsa → yanlış ürün / fiyat gelir

---

### ✅ DOĞRU DEBUG

Şunu yaz:

```js
const offerings = await Purchases.getOfferings();
console.log(JSON.stringify(offerings, null, 2));
```

👉 Bak:

* `current` hangi offering?
* içindeki `product.identifier` ne?

### Kontrol:

* Xcode → Scheme → Run → Options
* StoreKit config **kapalı olsun**
